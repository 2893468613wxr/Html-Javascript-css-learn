CSS图片样式

CSS图片样式主要包括图片的宽度、高度、边框、边框颜色、背景、透明度、边距、内边距、外边距等。

1. 图片宽度和高度

通过width和height属性可以设置图片的宽度和高度。width属性设置图片的宽度，height属性设置图片的高度。单位可以是px、em、%等。

2. 图片边框

通过border属性可以设置图片的边框。border属性可以设置边框的宽度、样式、颜色。border-width属性设置边框的宽度，border-style属性设置边框的样式，border-color属性设置边框的颜色。

3. 图片边框颜色

通过border-color属性可以设置图片的边框颜色。border-color属性可以设置边框的颜色，可以设置单一边框的颜色，也可以设置四个边框的颜色。

4. 图片背景

通过background属性可以设置图片的背景。background属性可以设置背景的颜色、图片、位置、重复方式等。

5. 图片透明度

通过opacity属性可以设置图片的透明度。opacity属性可以设置图片的透明度，取值范围是0~1，0表示完全透明，1表示完全不透明。

6. 图片边距

通过margin属性可以设置图片的边距。margin属性可以设置四个方向的边距，也可以设置单个方向的边距。

7. 图片内边距

通过padding属性可以设置图片的内边距。padding属性可以设置四个方向的内边距，也可以设置单个方向的内边距。

8. 图片外边距

通过margin属性可以设置图片的外边距。margin属性可以设置四个方向的外边距，也可以设置单个方向的外边距。
